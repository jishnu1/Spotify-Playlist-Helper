1. install Java
2. download app.jar and text files to a folder
3. open Spotify desktop application
4. select (ctrl A), copy (ctrl c), and paste (ctrl v) Spotify library into LIBRARY.txt
5. rename lists inside PLAYLISTS.txt (no spaces) and add more if needed
6. rename list files using the same names
7. select (ctrl A), copy (ctrl c), and paste (ctrl v) each Spotify playlist into each list file
8. use command prompt to enter this line inside the folder:
   java -jar app.jar
